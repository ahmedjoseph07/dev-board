Question-1: What are the different ways to select an element in the DOM?

getElementById("id"):-Selects a single element by its ID.
getElementsByClassName("class"):-Selects multiple elements with the same class.
getElementsByTagName("tag"):-Selects elements by tag name.
querySelector("selector"):-Selects the first element that matches a CSS selector.
querySelectorAll("selector"):-Selects all elements that match a CSS selector (NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:- Gets the HTML content inside an element, including tags.
innerText:- Retrieves only the visible text, ignoring hidden elements
textContent:- Gets or sets all text inside an element, including hidden text, but cannot parse HTML tags.

Question-3: What is event delegation in the DOM?

event delegation:- When a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

event bubbling:- When an event starts from the target element and propagates up to its parents.

Question-5: How do you create, add, and remove elements using JavaScript?

create an element:- document.createElement("tag").
add it to the DOM:- use parentElement.appendChild(newElement).
remove an element:- use parentElement.removeChild(element).
